# Evaluador de expresiones LISP

## Construir

```bash
./gradlew clean build
```

## Probar

```bash
./gradlew test
```

## Reporte de cobertura

```bash
./gradlew jacocoTestReport
xdg-open build/reports/jacoco/test/html/index.html
```

## Correr

```bash
./LispExpr
```

## Log de errores

```bash
less err.log
```
